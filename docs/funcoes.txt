Página Inicial: Filtro de animais e feed de publicações com paginação (máximo de 20 animais)

Perfil do usuário:
- Animais cadastrados: Mostrar animais cadastrados pelo usuário, com botões para editá-lo e excluí-lo.
- Alterar informações do animal: Formulário para alterações das informações do animal e possibilidade para alterar o "motivo para doação"(
caso seja um animal para doação).

	!!!! Excluir o "motivo para doação" do animal quando ele for adotado e mostrar um alerta ("o motivo será excluído") quando isso acontecer.

- Alterar informações pessoais: Formulário para alterar as informações usuário (exceto e-mail, senha e username).
- Informações pessoas: Tela para apresentar as informações do usuário.
- Excluir conta: Tela para certificar a exclusão.
- Sair: Sair da sessão. 

Perfil do animal: Imagem do animal, setor com informações sobre o animal, mapa com sua possível localização e comentários dos usuários da página.

Login: Tela de login (Login do usuário).

Cadastro do Animal: Tela de cadastro, com reCaptcha.

--> Animais perdidos:

- Caso 1: O animal é cadastrado pelo seu dono.

- Caso 2: O animal é cadastrado por alguém da rua.

(
	nome recebe value "sem nome";
	imagem recebe default "unknown.jpg";
	categoria só apresentará os "campos perdido" e "para adoção" disponíveis;
	acrescentar sexo desconhecido;
	raça recebe default "não informada";
	vacinado, castrado e vermifugado serão um select (sim, não, não informado);
	tipo recebe ('gato/cachorro');
)

--> Animais para adoção:

- Caso 1: O animal é cadastrado por uma Ong.

- Caso 2: O animal é cadastrado pelo dono que não pode cuidar mais do animal.


(
	O usuário precisará informar o motivo pelo qual está cadastrando o animal (um comentário);
)


Cadastro: Tela de cadastro, com reCaptcha(Cadastro do usuário).

Biografia: Tela para cadastrar biografia(valor padrão "Não cadastrado").

Doação: Tela de doação (Mensagens para incentivar a doação e um link para o site vakinha).

Entre em contato: Tela para entrar em contato com os gerenciadores da página (Nome e email opcionais).

Termo de uso: Tela mostrando o termo de uso da página.

Polítcas de privacidade: Tela mostrando as políticas de privacidade da página.

Perguntas frequentes: Tela mostrando as possíveis respostas para as perguntas mais realizadas.

Sobre: Tela apresentando o site, o grupo e parcerias.

Incentivação: Tela com mensagens de incentivo a adoção, endereços indicando os locais para essa ação e depoimentos dos usuários(incluir paginação).


OBS: cada usuário só poderá ter um depoimento, ele aparecerá no topo dos depoimentos, sendo permitido sua edição.
